
# Saint Judy - Judy's Jokes

## Setup in Vercel

Add environment variables:
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY

- / (home)
- /categories (browse all categories)
- /[category] (dynamic category pages)
